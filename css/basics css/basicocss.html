Basicos CSS

Selectores

  • "*" es el selector universal,selecciona a todos los elementos
  • Los selectores de tipo se llaman con el nombre del elemento(tagg)
  • Selector por clase,se le agrega una clase al tag, y luego se llama con . y la clase
  • Selector por id, igual que el de clase pero se pone un id xd, se llama con #
  • Por atributo: se crea un atributo, se llama el atributo entre [] de la misma forma q esta escrito en el tag
  • Para seleccionar de forma descendiente, se llama contenedor y luego el siguiente tag, ej: h2 p. Esto se puede hacer con classes, ids de la misma forma
  • Pseudo clases se llaman con :, una puede ser hover que es para cuando paso el mouse por encima
